Drop your final designer logo files in this folder with these exact names:

- elecon-logo-dark.png
- elecon-logo-light.png

Then run from project root:

python scripts/swap_logo_assets.py

This will replace logo files in:

- frontend/public/elecon-logo-dark.png
- frontend/public/elecon-logo-light.png
- backend/app/assets/elecon-logo-dark.png
- backend/app/assets/elecon-logo-light.png

No code changes are needed after swapping.
